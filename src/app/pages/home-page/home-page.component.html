Welcome to the Friends demo App!

This app demonstrate how to create a new angular app from scratch. The main requirement is: Create an input form using Material to insert multiple “friends” with the following fields:

  • Name
  • Age
  • Weight
  • Friends

Acceptance conditions are as follows (ACs):

  • Use the current version of Angular
  • Be production ready
  • Use Angular Material
  • Use NgRx for state management
  • Display results using D3.Js library
  • Make your components stateless
  • Use of git and organization of commits
  • Provide unit test coverage (testing actions, reducers, selectors and pipes)
  • The code should be stable, maintainable, and bug free.

Due to time constraints a minimum viable product (MVP) version that meets the requirements was created. It showcases different technologies and features of the Angular framework. The following are pending tasks that could help further increase the quality of this demo:

  • Security: Secure the route access using guards and integrate with an identity system or security code from the back-end to control authentication and authorization.
  • Localization: Provide translation of the UI to different languages/region. Support international formats where required.
  • Responsiveness: The responsiveness of the app needs more work
  • Test coverage: Since this is a demo app only basic coverage is provided. In a truly production ready a higher level is desired Different automated tests to implement can be: unit, component, DOM and e2E tests. Performance testing is also advisable.
  • Improvements to the graphics: Simple bar charts are provided. More complex one would be required for an actual app. The graphics would need to be responsive and interactive whenever possible, updating them by transitions instead of by refreshes.
  • Additional code refactor: Improve the existing code by creating new components to extract reusable code, reduce the size of the larger classes, and moving common files to shared folders/components. Style files could use some cleaning to reuse common styles used in different components.
